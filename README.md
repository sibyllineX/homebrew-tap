# SibyllineX Tap

## How do I install these formulae?

`brew install sibyllinex/tap/<formula>`

Or `brew tap sibyllinex/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sibyllinex/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
